# Hindu YUVA at UMB - Official Website

![Hindu YUVA Logo](https://via.placeholder.com/200x100/FF6B35/ffffff?text=Hindu+YUVA)

## 🕉️ About Hindu YUVA

**Youth for Unity, Virtue and Actions!** 

Hindu YUVA at UMass Boston is dedicated to promoting Hindu culture, values, and community. We create a vibrant and inclusive platform for students to celebrate, promote and engage with Hindu culture, traditions and values.

## 🌟 Features

- **Modern, Responsive Design** - Works perfectly on all devices
- **Cultural Events Management** - Showcase upcoming events and celebrations
- **Gallery** - Share photos from cultural events and activities
- **Resource Hub** - Academic and career advising resources
- **Temple Directory** - Local temple information and directions
- **Contact & Join** - Easy way to get involved with the community

## 🚀 Quick Start

### Prerequisites
- Git installed on your computer
- GitHub account
- Vercel account (free)

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/hindu-yuva-umb.git
   cd hindu-yuva-umb
   ```

2. **Open in your browser**
   ```bash
   # Simply open index.html in your browser
   # Or use a local server:
   python -m http.server 8000
   # Then visit: http://localhost:8000
   ```

## 🌐 Deployment

### Deploy to Vercel (Recommended)

1. **Push to GitHub**
   ```bash
   git add .
   git commit -m "Initial commit"
   git push origin main
   ```

2. **Connect to Vercel**
   - Go to [vercel.com](https://vercel.com)
   - Sign up/Login with GitHub
   - Click "New Project"
   - Import your GitHub repository
   - Deploy!

3. **Custom Domain** (Optional)
   - In Vercel dashboard, go to Settings → Domains
   - Add your custom domain
   - Update DNS settings as instructed

### Alternative Deployment Options

- **Netlify**: Drag and drop the folder to Netlify
- **GitHub Pages**: Enable in repository settings
- **Firebase Hosting**: Use Firebase CLI
